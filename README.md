# Bus Reservation System (C)

## Description
A console-based bus reservation system that allows users to book and cancel tickets, while admins manage bus data.

## Features
- User login & registration
- Role-based access (admin/user)
- Ticket booking & cancellation
- View booking history
- Bus management system

## Technologies
- C Programming
- File Handling
- Structures

## How to Run
gcc main.c
./a.out
